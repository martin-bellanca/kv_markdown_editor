Para generar la imagen PNG del diagrama:

  1. Instalar Graphviz:
     sudo dnf install graphviz

  2. Ejecutar el script:
     ./generate_diagram.sh

  3. Ver el resultado:
     xdg-open md_document_editor_v2_class_diagram.png
