# Teclas rápidas — Editor de Markdown

Resumen de atajos del editor. Hay dos modos:

- **Visualización** — navegás y seleccionás líneas.
- **Edición** — estás escribiendo dentro de una línea (con el cursor de texto).

---

## Navegación (visualización)

| Tecla | Acción |
|---|---|
| ↑ / ↓ | Línea anterior / siguiente |
| Re Pág / Av Pág | Subir / bajar una página |
| Ctrl+Inicio / Ctrl+Fin | Ir al inicio / fin del documento |
| Ctrl+↑ / Ctrl+↓ | Título anterior / siguiente (cualquier nivel) |
| Ctrl+Shift+↑ / Ctrl+Shift+↓ | Título anterior / siguiente del **mismo nivel** |
| Alt+Shift+↑ / Alt+Shift+↓ | Título **padre** (arriba / abajo en la jerarquía) |

---

## Entrar y salir de edición

| Tecla | Acción |
|---|---|
| Enter **o** F2 | Editar la línea seleccionada (cursor al final) |
| Click en la línea ya seleccionada | Editar (cursor donde hacés click) |
| Enter / F2 (editando) | Confirmar y salir de edición |
| Esc | Cancelar la edición (descarta los cambios de esa línea) |

> La navegación por títulos (Ctrl+↑↓, etc.) también funciona **editando**: salta al
> título manteniendo el modo edición y la columna del cursor.

---

## Edición de texto entre líneas

| Tecla | Acción |
|---|---|
| ↑ / ↓ | Mover la edición a la línea de arriba / abajo (mantiene la columna) |
| ← / → | En el borde de la línea, pasar a la anterior / siguiente |
| Enter | Partir la línea en el cursor (el resto baja a una línea nueva) |
| Shift+Enter | Insertar una línea vacía **arriba** y editarla |
| Retroceso (al inicio) | Unir con la línea de arriba |
| Supr (al final) | Unir con la línea de abajo |
| Alt+↑ / Alt+↓ | Mover la línea arriba / abajo |

---

## Selección múltiple (visualización)

Selección **contigua** (un bloque de líneas seguidas), para aplicarle acciones.

| Tecla | Acción |
|---|---|
| Shift+↑ / Shift+↓ | Extender la selección hacia arriba / abajo |
| Shift+Click | Extender la selección hasta la línea clickeada |
| Esc | Limpiar la selección (vuelve a una sola línea) |

> Estando en edición, **Shift+↑↓** o **Shift+Click** salen a visualización y
> empiezan a seleccionar.

---

## Acciones sobre el bloque seleccionado

| Tecla | Acción |
|---|---|
| Supr | Borrar las líneas seleccionadas |
| Alt+↑ / Alt+↓ | Mover el bloque arriba / abajo |
| Ctrl+C / Ctrl+X / Ctrl+V | Copiar / cortar / pegar |
| Ctrl+D | Duplicar el bloque |
| Tab / Shift+Tab | Indentar / desindentar |
| Ctrl+T | Convertir a tarea ⇄ texto |
| Ctrl+L | Convertir a lista ⇄ texto |

> **Pegar:** con **una** línea seleccionada, pega **debajo**; con **varias**,
> **reemplaza** el bloque. También pega texto copiado de otras aplicaciones.

---

## Tareas (checkbox)

| Tecla | Acción |
|---|---|
| Ctrl+Espacio | Marcar / desmarcar. En visualización afecta al bloque; editando, a la línea |
| Ctrl+Click | Marcar / desmarcar la tarea clickeada |

> El toggle de un bloque: si **todas** las tareas están marcadas, las desmarca;
> si hay alguna sin marcar, las marca todas.
